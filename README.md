# Data Cleaning & Reporting Automation

## Project Overview
This project automates data cleaning and reporting workflows using Excel, Power BI, and Python. The dataset was cleaned by removing duplicates, handling missing values, and standardizing data formatting. Interactive dashboards were created in Power BI for sales analysis and reporting.

## Tools Used
- Excel
- Power BI
- Python
- Pandas
- Openpyxl

## Features
- Remove duplicate records
- Handle missing values
- Create TotalSales column
- Automated data cleaning using Python
- Interactive Power BI dashboard

## Files Included
- sales_data.xlsx
- Cleaned_Output.xlsx
- clean.py
- Sales_Report_Dashboard.pbix
- README.md

## How to Run
Install required libraries:

```bash
pip install pandas openpyxl
```

Run Python script:

```bash
python clean.py
```

## Dashboard Features
- Total Sales KPI
- Sales by City
- Product Revenue Analysis
- Payment Method Distribution
- Sales Trend Analysis
- Interactive Filters
